Leap

Leap is an app for learning the English alphabet and pronunciation using Google speech recognition. First, you will learn how to read and write the English alphabet with the help of audio recordings, the international phonetic alphabet, and your native alphabet. Next, you will begin the Basic English unit with a curated list of vocabulary words and reading exercises. Google speech recognition will evaluate your pronunciation results, which is followed by a quiz. Currently available in Korean.

Main Activity

Vocabulary Lesson

Alphabet Activity

Handwriting Choices

Users are presented with a grid of letters to practice their handwriting. The Handwriting Activity will open when a letter is selected.

Alphabet List

Users can study the sounds of the alphabet alongside their native alphabet transliteration. If the speaker button is pressed, a recording of a human’s pronunciation is played from the Raw folder.

Alphabet Flashcards

Users can can study the sounds of the alphabet using a flashcard format that includes both the international phonetic alphabet and their native alphabet transliteration.

Handwriting

Users can practice their handwriting using a simple paint tool. An erase button is provided.

Vocabulary Lesson

Vocabulary Words Exercise

The vocabulary exercise uses a text to speech engine to speak the English vocabulary word. Users can study using a list or flashcards.

Reading Exercise

The reading exercise presents select vocabulary words in a functional context. A text to speech engine speaks both the English and native language translation.

Pronunciation Exercise

The pronunciation exercise uses a SpeechRecognizer intent to evaluate the user’s pronunciation using an arraylist of confidence scores. User is given a grade of correct, close, or incorrect.

Quiz Exercise

The quiz exercise tests the user on their comprehension of the vocabulary words. An enum value tracks the “attempt” (first try or second try) to accurately score the user. A hint is available.

Architectural Design

Data Structure

The data structure of Leap models the hierarchical structure of a language book: an ArrayList of Units, Lessons, Exercise, ExerciseItems is passed to each new Activity or Fragment.

User Progress

“Completable” and “Answerable” interfaces allow the app to track user progress after a user interacts with an item, which is then persisted using SharedPreferences.

Locale

The data that is responsible for creating units is kept in locale-specific JSON file in the asset folder (for example, data-ko.json). User-facing strings, such as alert dialog messages, are kept in the String Resources. On first launch, the user is prompted to select their native language, which will load the appropriate asset file and set the locale with a configuration change.

Speech Analysis

The speech analysis works by using a SpeechRecognizer intent that will call a startActivityForResult method. An arraylist of of the SpeechRecognizer’s confidence scores (0-100% confidence) is used with a string contains method to assess whether a user it correct, close, or incorrect.
